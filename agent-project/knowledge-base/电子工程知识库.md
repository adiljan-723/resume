## 电路分析

### 基尔霍夫定律
基尔霍夫电流定律（KCL）：在集总电路中，任一时刻流入任一节点的电流代数和为零。
数学表达：∑I_in = ∑I_out
基尔霍夫电压定律（KVL）：在集总电路中，任一时刻沿任一闭合回路的电压降代数和为零。
数学表达：∑V_drop = 0

### 叠加定理
在任何线性电路中，多个独立源同时作用产生的响应（电压或电流），等于每个独立源单独作用时产生的响应之和。
应用步骤：
1. 每次只保留一个独立源，其他电压源短路、电流源开路
2. 分别计算各独立源单独作用时的响应
3. 将所有响应代数相加

### 戴维宁定理
任何线性有源二端网络，对外电路而言，可以等效为一个理想电压源与一个电阻串联的支路。
- 戴维宁等效电压 Vth：端口开路电压
- 戴维宁等效电阻 Rth：内部独立源置零后的端口输入电阻

### 一阶RC电路
时间常数 τ = RC
充电：v(t) = V0(1 - e^(-t/τ))
放电：v(t) = V0 · e^(-t/τ)
MATLAB验证代码：
```matlab
R = 1000; C = 1e-6; tau = R*C;
t = 0:tau/100:5*tau;
v_charge = 5*(1 - exp(-t/tau));
plot(t*1000, v_charge);
xlabel('时间 (ms)'); ylabel('电压 (V)');
title('RC充电曲线');
```

## 信号处理

### 傅里叶变换
时域信号 → 频域表示的数学工具。
连续傅里叶变换：F(ω) = ∫f(t)e^(-jωt)dt
FFT（快速傅里叶变换）是离散傅里叶变换的高效算法。

MATLAB示例：
```matlab
fs = 1000; t = 0:1/fs:1-1/fs;
x = sin(2*pi*50*t) + 0.5*sin(2*pi*120*t);
X = fft(x);
f = (0:length(X)-1)*fs/length(X);
plot(f(1:length(f)/2), abs(X(1:length(f)/2)));
```

### 采样定理
奈奎斯特-香农采样定理：要从采样信号中无失真恢复原始连续信号，
采样频率 fs 必须大于信号最高频率 fmax 的 2 倍。fs > 2fmax

### 卷积
两个信号 f(t) 和 g(t) 的卷积定义为：
(f * g)(t) = ∫f(τ)g(t-τ)dτ
生活类比：用湿毛巾擦过桌面，毛巾带走污渍的过程就是卷积。

## 单片机开发

### STM32 GPIO 初始化（标准库）
```c
#include "stm32f10x.h"

void GPIO_Init(void) {
    GPIO_InitTypeDef GPIO_InitStructure;
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA, ENABLE);
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_0;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_PP;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
}
```

### 中断优先级
STM32使用NVIC（嵌套向量中断控制器）管理中断。
优先级分组：抢占优先级 + 子优先级
抢占优先级高的可以打断抢占优先级低的中断服务程序。

### 常见调试坑点
1. 忘记开启外设时钟（RCC_APB2PeriphClockCmd）
2. GPIO模式配置错误（推挽/开漏混淆）
3. 中断服务函数名拼写错误（必须与启动文件中的向量表一致）
4. 延时函数未初始化SysTick

## 通信原理

### 调制方式
AM（调幅）：载波幅度随调制信号变化
FM（调频）：载波频率随调制信号变化
PM（调相）：载波相位随调制信号变化

### 5G NR 基础
- 频率范围 FR1：450MHz - 6GHz
- 频率范围 FR2：24.25GHz - 52.6GHz（毫米波）
- 子载波间隔：15/30/60/120 kHz
- OFDM符号长度与子载波间隔成反比
