# 测试用例 — 验证 Agent 工作流

## 用例 1：电路分析（类别1）
**输入：** "设计一个截止频率1kHz的低通RC滤波器，计算R和C的值"

**期望路由：** 意图识别→1→知识库检索→电路分析专家链
**期望输出包含：** 时间常数公式、R/C计算过程、MATLAB验证代码、Bode图建议

---

## 用例 2：信号处理（类别2）
**输入：** "帮我理解FFT的频谱泄漏问题，怎么加窗解决？"

**期望路由：** 意图识别→2→知识库检索→信号处理专家链
**期望输出包含：** 频谱泄漏的物理解释、常见窗函数对比（Hamming/Hanning/Blackman）、MATLAB代码

---

## 用例 3：单片机开发（类别3）
**输入：** "STM32怎么配置外部中断？给我一个按键检测的完整代码"

**期望路由：** 意图识别→3→知识库检索→单片机专家链
**期望输出包含：** 硬件连接图描述、GPIO+EXTI+NVIC配置代码、防抖处理、调试技巧

---

## 用例 4：知识库无法覆盖（兜底测试）
**输入：** "量子计算机什么时候能替代经典计算机？"

**期望路由：** 意图识别→不属于1-5→通用学习帮助链
**期望行为：** 诚实说明这不是本助教的专长领域，建议咨询量子物理方向

---

## 用例 5：边界测试（混合问题）
**输入：** "我的单片机项目需要做FFT频谱分析，STM32F103能跑得动吗？"

**期望路由：** 意图识别→3或2→检索→对应专家链
**期望输出包含：** STM32F103的算力评估、CMSIS-DSP库使用建议、或建议上STM32F4/H7

---

## 验证清单
- [ ] 意图识别准确率 > 85%（5个测试用例至少4个正确路由）
- [ ] 知识库检索命中率 > 60%（前3个用例题应命中）
- [ ] 回答包含"逐步推理"结构
- [ ] 代码片段可运行
- [ ] 对话记录保存在Dify日志中
