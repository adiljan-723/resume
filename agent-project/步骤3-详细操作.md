# 步骤3：导入工作流 — 超详细版

---

## 3.1 进入 Dify 工作室

1. 浏览器打开 **https://cloud.dify.ai**
2. 左上角看到 Dify 的 logo 和你的头像
3. 点击左侧菜单栏的 **「工作室」**（图标是一个方块 + 笔）
4. 现在你看到的是应用列表页

---

## 3.2 创建 Chatflow 应用

5. 页面中间或右上角找到 **「创建空白应用」** 按钮（蓝色大按钮）
6. 点击后会弹出一个选择框，有三个选项：
   - 聊天助手
   - **Chatflow** ← 选这个
   - 工作流
7. 点击「Chatflow」后，出现输入框：
   - **应用名称** 填：`智能电子工程学习助手 Agent`
   - **描述** 填下面这段话（直接复制粘贴）：

   ```
   基于RAG+多工具调用的电子工程AI助教。集成5领域意图识别（电路/信号/单片机/通信/学习），4条专家Agent链，混合检索知识库，支持MATLAB/C代码生成，覆盖深大电子信息工程核心课程。
   ```

8. 点击 **「创建」** 蓝色按钮

---

## 3.3 清空画布

9. 创建后会进入一个画布（白色背景，带网格点）
10. 画布上默认有 3 个节点连在一起：
    - `[Start]` → `[LLM]` → `[Answer]`
11. 把鼠标移到画布空白处，按住左键拖拽框选所有 3 个节点
12. 按键盘上的 **Delete** 键（或 Backspace）
13. 画布变空白 ✅

---

## 3.4 导入 DSL 文件

14. 找到画布上方的工具栏，点击 **「导入 DSL 文件」** 按钮
    （一般在画布左上角附近，图标是一个文件夹+箭头）
    
    > 💡 如果找不到这个按钮，在画布空白处右键 → 点击「导入 DSL」

15. 弹出一个文件选择窗口，点击 **「上传文件」**
16. 找到这个文件并双击打开：
    ```
    C:\Users\86178\agent-project\dify-workflow.yml
    ```
17. 上传完成后，点击弹窗右下角的 **「导入」** 按钮
18. 画布上会出现一堆节点和连线 ✅

---

## 3.5 修复知识库节点（红色警告①）

19. 导入后，**「知识库检索」** 这个节点是红色的（缺少知识库绑定）
20. 点击这个红色节点，右侧弹出设置面板
21. 在设置面板中找到 **「知识库」** 下拉框
22. 点击下拉框，选择 `电子工程教材知识库`（你第2步创建的）
23. 确认下面几个参数：
    - 检索方式 → 选 `混合检索`
    - Top K → 填 `5`
    - 分数阈值 → 填 `0.6`
    - Rerank → 开关打开 ✅
24. 红色消失 ✅

---

## 3.6 配置模型 API Key

25. 鼠标移到页面**右上角你的头像**，点击
26. 下拉菜单中点击 **「设置」**
27. 左侧菜单点击 **「模型供应商」**
28. 你会看到一长串模型供应商列表（OpenAI、DeepSeek、Anthropic 等）

### 方案A：用 DeepSeek（推荐，便宜）

29. 找到 **DeepSeek** → 点击
30. 点击 **「添加」** 按钮
31. API Key 输入框出现
32. 去另一个浏览器标签页打开：**https://platform.deepseek.com**
33. 注册/登录 DeepSeek
34. 点击左侧「API Keys」→「创建新的 API Key」
35. 复制生成的 Key（长字符串）
36. 回到 Dify 标签页，粘贴到输入框
37. 点击 **「保存」**

### 方案B：用 OpenAI（效果好，略贵）

29. 找到 **OpenAI** → 点击
30. 点击 **「添加」**
31. 去 **https://platform.openai.com/api-keys** 获取 Key
32. 粘贴到 Dify，保存

---

## 3.7 修改节点的模型选择

38. 关闭设置面板，回到画布
39. 逐个点击每个 LLM 节点，在右侧面板把模型从 `gpt-4o` 改成你配置的模型：
    - 如果用 DeepSeek → 选 `deepseek-chat`
    - 如果用 OpenAI → 选 `gpt-4o-mini`（分类和整合节点）和 `gpt-4o`（4个专家节点）

| 节点名称 | 推荐模型 |
|----------|----------|
| 意图识别 | deepseek-chat 或 gpt-4o-mini |
| 电路分析专家 | deepseek-chat 或 gpt-4o |
| 信号处理专家 | deepseek-chat 或 gpt-4o |
| 单片机开发专家 | deepseek-chat 或 gpt-4o |
| 通信原理专家 | deepseek-chat 或 gpt-4o |
| 学习规划助手 | deepseek-chat 或 gpt-4o-mini |
| 回答整合 | deepseek-chat 或 gpt-4o-mini |

---

## 3.8 设置开场白

40. 画布上方工具栏找到 **「应用设置」** 按钮（齿轮图标）
41. 找到 **「对话开场白」** 区域
42. 粘贴以下内容：

```
👋 你好！我是电子工程学习助手，深圳大学电信学院AI助教。

我可以帮你：
⚡ 电路分析 — 模拟/数字电路、放大器、滤波器
📊 信号处理 — FFT、频谱分析、采样、MATLAB
🔌 单片机开发 — STM32、C语言、传感器、中断
📡 通信原理 — 调制解调、高频电路、5G
📚 学习规划 — 课程规划、考研指导、论文写作

抛出你的问题吧！👇
```

43. 点击右上角 **「发布」** 蓝色按钮
44. 弹出确认框 → 点击 **「发布」**

---

## 🎉 完成！

发布后会生成一个 URL（类似 `https://cloud.dify.ai/app/xxxxx`），这个链接就是你的 Agent 访问地址。
